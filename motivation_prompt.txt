[TASK TYPE]  
사용자의 발화(텍스트)와 사진(이미지)을 기반으로 구독 동기 8개 카테고리 중 1개로 분류하기.  

[INSTRUCTIONS]  
1. 입력된 텍스트와 이미지를 함께 분석한다.  
2. 사진 속 요소(색감, 분위기, 사람, 패션, 제품 등)를 고려하여 구독 동기를 추론한다.  
3. 반드시 아래 8개 카테고리 정의 중 하나를 선택한다.  
4. 선택 근거를 1~2문장으로 설명한다.  
5. 최종 출력은 반드시 JSON 형식으로 제공한다.  

[DO]  
- 텍스트와 사진 둘 다 고려  
- 반드시 실용소비, 웰니스, 프리미엄, 감성, 자기계발, 유머, 비주얼, 라이프스타일 중 하나만 선택  
- 카테고리명은 반드시 정의된 단어 그대로 사용
- 출력 형식:  
```json
{ "구독동기": "카테고리명", "근거": "설명" }

[DON'T]
- 중복 선택 금지
- 새로운 카테고리 생성 금지
- 텍스트 또는 이미지를 무시하지 말 것

[EXAMPLES]
- 입력(텍스트): "이 계정은 매번 장보는 꿀팁을 알려줘서 꼭 챙겨봐요."
- 입력(이미지): 장바구니에 가득 담긴 식료품 사진
{ "구독동기": "실용소비", "근거": "생활에 도움이 되는 정보와 절약 팁 제공, 장보기 관련 이미지와 일치" }
- 입력(텍스트): "사진 색감이 너무 예뻐서 그냥 보는 것만으로 힐링돼요."
- 입력(이미지): 노을이 지는 바다 사진
{ "구독동기": "감성", "근거": "시각적 색감과 분위기 요소가 감성적 반응을 유발하기 때문" }

[CONTENT]
구독 동기 정의 (팔로워 관점):
1. 실용소비: 일상에 도움이 되는 정보, 돈/시간 절약 팁을 얻기 위해 구독
2. 웰니스: 건강한 생활 방식, 친환경 소비에 대한 관심으로 구독
3. 프리미엄: 고급 브랜드나 하이엔드 제품에 대한 관심, 소유에 대한 동경
4. 감성: 색감, 분위기, 영상미 등 시각적인 감성에 반응해 구독
5. 자기계발: 공부, 루틴, 커리어 등 성장 동기나 동기부여를 얻기 위해 구독
6. 유머: 웃기고 가볍게 볼 수 있는 콘텐츠로 스트레스를 해소하기 위해 구독
7. 비주얼: 외모, 스타일, 패션 등 보는 것 자체로 자극을 주는 콘텐츠를 좋아함
8. 라이프스타일: 나와 비슷한 일상, 현실적인 공감에서 위로를 얻고 싶어 구독
